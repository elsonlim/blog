CSS variable allows you to create a custom name to a value a good place to put them is in body or :root

:root { --content-background: blue; }

To use the variable, we need the var() function

code { color: var(--content-code); }

we can override the variables on specific class

body.dark {
  --header-background: black;
  --header-text: white;
  --content-text: white;
  --content-background: brown;
  --content-code: lightSeaGreen;
}

We cam now toggle the class to change all the colors within the page.

function toggleDarkMode() {
  const element = document.getElementsByTagName("body")[0];
  element.classList.toggle("dark");
}
